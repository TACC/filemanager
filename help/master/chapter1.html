Introduction

The XSEDE File Manager(XFM) is a Java Applet designed to allow XSEDE users to manage their scientific data across heterogeneous resources both internally and externally to the XSEDE.

 

Intended Use

 

The XFM is intended to complement existing file management utilities such as RFT, globus-url-copy, scp, etc. For users accustomed to traditional desktop file managers, the XFM should serve as a convenient bridge from their desktop experience to grid-based file management.  Features such as:

 

  • Drag-and-drop copying
  • Single sign-on authentication
  • Status monitoring
  • Historical logging
  • Search
  • File sharing and collaboration

 

provide a convenient bridge between the desktop experience and grid technologies.
