Getting Started

 

Introduction

This section serves as a quick start guide to using the XSEDE File Manager. For further detail, please consult the chapters corresponding to the following categories.

 

Authenticating

The XFM uses your default XSEDE credentials stored in the portal. They are loaded when the XFM starts and used throughout the life of the session. If your credentials expire, please logout and log back in to the portal.

 

Discovering Resources

When the XFM starts, the left panel will automatically open up to your local home directory. The right panel will appear empty and prompt you to select a resource from the drop down combo box. By default the combo box is populated with all the resource on which you have accounts in the XSEDE. This list corresponds to the resources listed under “System Accounts” in the Accounts and Usage portlet.  Other resources can manually be added and removed via the Resource List.

 

Browsing Files

To browse files on a particular resource, open the Resource List by clicking on the XSEDE logo in the upper left of either browsing panel.  Double clicking on a resource will remove the Resource List and display a listing of the file in your home directory on that resource.  Once your home directory appears, you can navigate through your file system by double clicking on directories or pressing the “Go To” button and entering a location directly in the “Go To” dialog box.

Double clicking on a file will download it to your local home directory. (This can be changed in your Preferences dialog.  XFM->Preferences->Security) Dragging and dropping the file between two windows will copy the file from the directory listed in the first panel to the directory listed in the second panel.

To rename a file, right click on the file and select rename. A text dialog box will appear allowing you to enter the new file name.

To delete a file, either click the trash can icon in the top menu bar, or right click on the file name and select "Delete."

To create a new directory, right click in the directory window and select "New Directory."

Right click support is enabled on all files providing quick access to all the above features.

 

Viewing Transfer History / Logs

All your file transfers will automatically be recorded by the XUP infrastructure. When you start the XFM, your last 150 transfers will be listed in the table at the bottom of the client. From the right click menu on this table, you can repeat the transfer, delete the record(s), or refresh your history.  You can view your transfer history and logs by selecting View->Show Transfers and View->Show Logs from the top application menu.

 

Receiving Notifications

Notification support is built into the XFM via the XSEDE Notification Service. You can select to have an email sent to you when your file transfer completes by navigating to the "Tools" menu item and selection the "Options" menu item. This will open the user options dialog box. Select the "Notifications" tab and choose your preferred notification method from the drop down combo box.

By default, notifications are disabled in the XFM. They can be turned on for all transfers via the Options dialog, or individually via the right click menu in the file transfer history table.
