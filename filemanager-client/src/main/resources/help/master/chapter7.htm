Notifications

 

Introduction         

 

The XFM supports notification of file completions via the TG Notification Service. Notifications may be sent by IM, text message, or email. The following sections describe how to utilize this feature.

Setting Default Notification Type

 

By default, notification is disabled for all file transfers. To enable a particular type of notification, say email, for all transfers, open the "Options" dialog, click on the "Notificiations" tab and select the "EMAIL" option in the drop down box. This will result in an email being sent at the conclusion of every email initiated from that XFM for the remainder of the session. This can be turned off again by selecting the "NONE" option in the drop down box of the "Options" dialog.

 

Updating Contact Information

 

If you are not receiving notifications, chances are your contact information is not correct in the TGUP. Please go to Profile Editor and edit this information.

 

Adding Notifications

Often times, notification will not be desired, nor needed for every file transfer. In these instances, we can assign a single notification event for the desired file transfer by right clicking on the transfer in the history table and selecting "Email" and the desired notification type.

 

Removing Notifications

 

To clear all notifications for a particular file transfer, select the transfer in the history table, right click and select "Email" -> "Clear all" from the right click menu.
